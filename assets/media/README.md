# Media Assets Directory

Upload images/videos here for automatic display on website.

## Folder Structure
- home/ - Homepage hero and program cards
- gallery/ - Gallery page (5 categories)
- about/ - About page media
- admissions/ - Admissions page media
- academics/ - Academics page media
- contact/ - Contact page media

## Supported Files
Images: .jpg, .png, .webp
Videos: .mp4, .webm

## Usage
1. Upload media to correct folder
2. For gallery: Add filename to assets/js/gallery-items.js
3. For captions: Edit captions.txt in category folder
4. Commit changes

## Video Guidelines
- Max 100MB per file
- Recommended: 50MB, 720p, MP4 format

## Caption Format
filename.ext | EN: English text | HI: Hindi text

See MEDIA_UPLOAD_GUIDE.md for details.
