# School Documents

This directory should contain downloadable documents for the website.

## Required Documents:
- `vps-brochure.pdf` - School brochure with overview, facilities, fees, and contact info (both EN/HI)
- `admission-form.pdf` - Printable admission application form
- `prospectus.pdf` - Detailed school prospectus

## Document Specifications:
- Format: PDF
- File size: < 5MB each (optimize for web download)
- Ensure bilingual content (English and Hindi) where applicable
- Include school branding and contact information

## Note:
Documents should be updated annually or as information changes. Ensure all information is accurate and current before publishing.
